1. Création du dossier Source

Créer un dossier local pour votre projet sur votre ordinateur comme suit :

  • mkdir pour créer un dossier
  • ls pour vérifier que le dossier a été créé
  • cd pour aller dans le dossier créé

    • 2. création Repository

    Aller sur le site web de github et créer un nouveau repository

    Dans Github Desktop ajouter votre nouveau projet repo et cliquer sur Ouvrir avec VSC

    4. Initialisation

    Ouvrir ligne de commande et se positionner sur le bon dossier avec la commande cd et chemin absolue du dossier

    Dans powershell initialiser le repository avec git init. Vérifier que le fichier .git est bien ajouté dans le dossier.
